📝 Web Note → ClickUp

Connect your ClickUp account to send notes as subtasks. Everything stays in your browser — no server, no account required.

Step 1

Connect with your API token

Paste your ClickUp Personal API token. Find it in ClickUp under Settings → Apps → API Token (it starts with pk_).

Step 2

Choose where notes go

Pick the List that holds (or will hold) your parent task.

Step 3

Choose the parent task

Each note becomes a subtask of this task.
